Photogrammetry test 2

Using Meshroom, Mesh Mixer and Blender. I am going to try making a mesh from aerial photos. Clean it up and look around in VR. Last time I tried making one from photos of a statue. Let’s see how this turns out.

I have loaded an aerial photo dataset into Meshroom. Using the default photogrammetry workflow I’ll make a new mesh.

We are about half way done.


You can see the volume of the trees above the ground. I could see this being a flowing mesh with peaks and valleys. I’m not sure how well the details will look under the tree canopies.

Meshroom has finished rendering and we now have a mesh. Let’s see how it ended up.

This is looking pretty rough. There are probably ways this could be cleaned up. Using Mesh mixers’s analysis tools we can patch this up.


Now the waiting begins.

15 minutes later Mesh Mixer gives us this land scape.

I think it looks smoother than before. I wouldn’t really call this a forest. It kinda reminds me of jagged rocks.

After running the tool again about 8 more times. I end up with this mesh which looks more smoothed out. All this rendering time has given me a chance to read the Meshroom manual. It had info on some of the nodes in the program. I’m excited to start customizing my meshes.  

 let’s export to Blender and we can check this out in VR.

These trees have seen better days.

Looking from a distance It definitely looks cool. It captured the ground and the terrain well.. It definitely lost a lot of detail though. It’s hard to tell those were trees . I’ll try doing a small object with my own camera and see how it goes.
